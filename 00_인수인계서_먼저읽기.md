# Cowork 인수인계서 — SmartFarm Agent 엑셀 재건(경로 B)

> 이 문서는 Cowork 작업공간에서 **첫 지시로 그대로 붙여넣어** 작업을 시작하기 위한 브리핑이다.
> 목표: 검증된 계산 엔진(`smartfarm_engine.py`)의 로직을 **실동작 견적 엑셀 워크북**으로 이식하고,
> 원본 경제성 엑셀의 깨진 시트를 복구한다.

---

## 0. 배경 (한 문단)
스마트팜 진단·설계·시공·경제성 Agent를 구축 중이다. 4개 축의 계산 로직은 이미
`smartfarm_engine.py`에 순수 함수로 고정되고 회귀 테스트 11개를 통과했다(8건 실측 벤치마크 포함).
이제 이 로직을, 사용자가 GPT 없이도 쓸 수 있는 **입력→자동계산 엑셀**로 옮긴다.

## 2. 첫 번째 작업: 실동작 견적 워크북 신규 생성
`SmartFarm_견적계산기.xlsx`를 새로 만든다. 시트 구성:

| 시트 | 내용 | 엔진 대응 함수 |
|---|---|---|
| `입력` | 사업유형·작목·지역·면적·피복·적설심·풍속·목표온도 등 | (입력) |
| `설계_규격` | 적설심·풍속→충족 규격, 형식별 최소사양 | `select_specs` |
| `설계_난방` | 최대난방부하·난방기용량·연료소비, 실측 이중검증 | `heating_load`,`verify_heating_vs_actual` |
| `시공_견적` | 공종별(C1~C13) 수량×단가 → 총공사비 | `apply_overheads` 외 |
| `시공_검증` | 총액÷면적 vs 8건 밴드 → 정상/경고 | `benchmark_check` |
| `경제성` | 생산량·매출·OPEX·손익·ROI/NPV/IRR/실질ROI | `production_kg`,`finance` |
| `기준데이터` | A-2 규격표·A-11 밴드·A-5 발열량 (참조 상수) | 상수 테이블 |

**핵심 규칙(엔진과 동일하게 강제):**
1. 골조는 '온실전체 개산(A-2 평단가)'과 '골조단독(A-8)'을 **분리 셀**로. 혼용 금지(이중계산 방지).
2. 벤치마크 검증 셀은 피복종류에 따라 밴드 자동 선택(필름115~220 / 유리180~230 / 불소230~310 천원/㎡).
3. 시세성 값(단가·노임·유가)은 `입력` 시트의 **수동 입력칸**으로 두고 주석에 출처(KAMIS·대한건설협회) 명시. 엑셀이 자동조회하지 않음.

**검증:** 완성 후 `입력`에 원채원 케이스(유리·3456㎡·702,030,000원·단가2500·적합도95·보조50%)를 넣어
`run_report.py` 데모 출력과 **동일 결과**(ROI 14.2%·Payback 7.1년·실질ROI 28.3%·밴드 정상)가 나오는지 대조.

## 3. 두 번째 작업: 원본 경제성 엑셀 복구
`1_경제성분석_..xlsx`는 **전체가 깨진 게 아니다.** 진단 결과 `#REF!` 오염은 3개 시트뿐:

| 시트 | #REF 수 | 조치 |
|---|---|---|
| `추정손익(월별)` | 1,820 | 참조링크 재연결(원본 참조 시트 추적) 또는 사업성 시트 구조로 재작성 |
| `FCF` | 4 | 4개 셀만 수식 복구 |
| `텐진vs쇼탕산` | 44 | 사용 안 하면 삭제, 쓰면 복구 |

**나머지 22개 시트는 정상**(사업성·전제조건·추정PL/CF·생산·매출·인건비·재료비·Capex 등).
→ `사업성` 시트가 FCF·ROI·NPV 구조의 정본. 이 구조를 기준으로 깨진 셀을 되살린다.

## 4. 작업 원칙 (이 프로젝트가 지켜온 것)
1. **데이터를 열어 확인하고, 실행해서 검증한다.** 엑셀 수식으로 옮긴 뒤 반드시 원채원 케이스로 재현 검증.
2. **엔진이 단일 계산 출처.** 엑셀 수식이 엔진과 다른 값을 내면 엔진이 맞다(테스트 통과분).
3. **근거 없는 값 금지.** 모든 단가·규격은 A/B/C 항목 근거. 추정은 주석에 `[추정]`+근거.
4. **8건 벤치마크는 회귀 기준.** 엑셀도 8건이 밴드 안에 들어와야 한다(test_engine.py의 ACTUALS).

```
그다음 3절에 따라 원본 경제성 엑셀의 깨진 3개 시트(#REF!)를 복구해줘.
```
